
## Useful CLI Tools to Pair With Claude Code

These CLI utilities pair well with Claude Code to supercharge your workflow:

| Tool | Purpose |
|------|---------|
| **[lazygit](https://github.com/jesseduffield/lazygit)** | TUI for Git |
| **[glow](https://github.com/charmbracelet/glow)** | Render Markdown in the terminal |
| **[llmfit](https://github.com/d有机산/llmfit)** | Find which models fit your hardware |
| **[models](https://github.com/jlowin/models-cli)** | List models, providers, and pricing |
| **[ranger](https://github.com/ranger/ranger)** | File explorer in the terminal |
| **[btop](https://github.com/aristocratos/btop)** | System resource monitor |
| **[csvlens](https://github.com/Youwotma/csvlens)** | View CSV files in the terminal |
| **[chafa](https://github.com/hpjansson/chafa)** | Display images in the terminal |
| **[z](https://github.com/rupa/z)** | Jump to frequently used directories |
